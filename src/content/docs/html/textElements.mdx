Los elementos de texto te permiten dar formato y significado específico a porciones de texto dentro de tus páginas web. Algunos enfatizan palabras clave, otros indican citas o código, y otros simplemente aplican estilos visuales.

## `strong`: Importancia y Relevancia (Énfasis Fuerte)

El elemento `<strong>` resalta texto que es importante o relevante dentro del contexto. Los navegadores suelen mostrarlo en negrita.

```html {"id":"01J3RTAS6W1EHS91ZK2W9CKA3B"}
Esta es una palabra <strong>muy</strong> importante.
```

Esta es una palabra muy importante.

## `em`: Énfasis y Acento
El elemento `<em>` enfatiza palabras o frases para darles un acento especial. Los navegadores suelen mostrarlo en cursiva.

```html {"id":"01J3RTE89766Q8TMQQQ2MZJ4EF"}
Este producto es <em>realmente</em> innovador.
```

Este producto es <em>realmente</em> innovador.

## `b`: Estilo Visual (Negrita sin Relevancia Semántica)
El elemento `<b>` simplemente aplica un estilo visual de negrita al texto, sin añadir un significado especial.

```html {"id":"01J3RTH0B4SJSP8WR0Z1881CJM"}
Este texto está en <b>negrita</b>.
```

Este texto está en <b>negrita</b>.

## i: Estilo Visual (Cursiva sin Relevancia Semántica)
El elemento `<i>` aplica un estilo visual de cursiva al texto, sin añadir un significado especial.

```html {"id":"01J3RV1R3XQRFTG53D0VA0BYQM"}
Este texto está en <i>cursiva</i>.
```

Este texto está en <i>cursiva</i>.

## u: Texto Subrayado
El elemento `<u>` subraya el texto, aunque su uso se ha vuelto menos común debido a la asociación con enlaces.

```html {"id":"01J3RV32HX74GDE5JEPJ1T9FTS"}
Este texto está <u>subrayado</u>.
```

Este texto está <u>subrayado</u>.

## s: Texto Tachado
El elemento `<s>` tacha el texto, indicando que ya no es relevante o válido.

```html {"id":"01J3RV4CZ8MX70FMY0ER1W7QHY"}
Este producto ya no está disponible <s>antes $50</s> ahora $30.
```

Este producto ya no está disponible <s>antes $50</s> ahora $30.

## `code`: Código y Caracteres Especiales
El elemento `<code>` muestra texto como código fuente o caracteres especiales, generalmente con una fuente monoespaciada.

```html {"id":"01J3RVA4QAJRHKKXP8K5BJNT0G"}
El código es <code>console.log("Hola mundo");</code>
```

El código es <code>console.log("Hola mundo");</code>

## `pre`: Formato Predeterminado
El elemento `<pre>` muestra texto con su formato original, incluyendo espacios, saltos de línea y tabulaciones. Es útil para mostrar código o poesía.

```html {"id":"01J3RVD7X0SAF2KSSJJAHCP0EB"}
<pre>
    function saludar() {
      console.log("¡Hola!");
    }
    </pre>    
```

## `blockquote`: Citas y Referencias Externas
El elemento `<blockquote>` indica que el texto es una cita o referencia externa. Los navegadores suelen mostrarlo con sangría.

```html {"id":"01J3RVKBDZ4MCVNSE809R29NZ3"}
<blockquote>
    "La imaginación es más importante que el conocimiento." - Albert Einstein
  </blockquote>  
```

<blockquote>
  "La imaginación es más importante que el conocimiento." - Albert Einstein
</blockquote>
